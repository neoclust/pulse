Member Function Documentation

addOptions (   $filename,
  $optionArray 
)

Add options that will be configurable through the page. All these options should be located in ONE .ini file. ($filename) If options for a module come from different files, call this method once for each file, passing the options in each file. optionArray is an array of ConfigurationOption objects.

Options are not added one by one but in an array, so that we can have only one XMLRPC request by concerned config file.
